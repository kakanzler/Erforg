`_` 始まりなので一覧には出ません。手で書くときの雛形です。
ブラウザから書く場合はこのファイルを触る必要はありません。

## 1冊ぶんの置き方

```
content/books/<本のslug>/
  _book.md            ← 本そのもの
  <記事のslug>.md     ← その本についての記事（何本でも）
```

## `_book.md`

```yaml
---
title: 本のタイトル
author: 著者名
category: 技術書 # 例: 技術書 / 小説 / ビジネス / 思想 …
tags: [タグ1, タグ2]
---
本そのものについてのメモがあればここに（省略可）。
```

## 記事

```yaml
---
title: 初読の記録 # 記事の見出し。本のタイトルとは別
dateRead: 2026-06-30 # 読了日 YYYY-MM-DD（ヒートマップの元）
rating: 4 # 0〜5 の整数（★の数）
tags: [設計, 習慣]
---

## 概要

どんな本か、一言で。

## 印象に残った点

- 心に残ったこと
- 引用など

## 学び

自分の言葉でのまとめ。
```

## ノート（本に紐づかないもの）

`content/notes/<カテゴリ>/<ノートのslug>.md`

```yaml
---
title: ノートのタイトル
date: 2026-06-30
tags: [タグ1]
---
本文。
```
